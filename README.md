
📊 Sales Analytics Dashboard (Flask + EDA)

A complete end-to-end Data Analysis project built using Python, where raw sales data is transformed into meaningful business insights and visualized through an interactive Flask dashboard.

🚀 Project Overview

This project focuses on analyzing a large sales dataset (~180,000+ records) to uncover trends, customer behavior, and product performance. The insights are then presented via a Flask-based web dashboard for easy interpretation.

🧰 Tech Stack
Python
Pandas, NumPy (Data Cleaning & Analysis)
Matplotlib (Visualization)
Flask (Web Dashboard)
HTML/CSS (Frontend)
📂 Dataset
Contains ~1.8 lakh rows of sales data
Includes:
Order Date
Product
Quantity Ordered
Price Each
Purchase Address
🔍 Key Features
📌 Data Processing
Cleaned missing and inconsistent data
Converted date formats and extracted features (Month, Hour, City)
Created new metrics like Revenue (Quantity × Price)
📊 Exploratory Data Analysis (EDA)
Monthly sales trends
Sales by city
Peak purchase hours
Product demand analysis
Frequently bought together products
💡 Business Insights
Identified peak sales months and hours for ad optimization
Discovered top-performing cities and products
Suggested product bundling opportunities
Highlighted high-revenue generating SKUs
🌐 Flask Dashboard
Interactive dashboard displaying:
KPIs (Total Revenue, Orders, etc.)
Sales trends
Product performance charts
Designed for real-time business insights
